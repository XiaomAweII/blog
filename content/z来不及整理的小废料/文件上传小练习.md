前端时间学习了文件上传, 用于上传图片, 学了富文本, 将图片生成富文本文件存入数据库当中, 然后我想到一个点子, 是我在写博客的过程中, 用vscode记笔记, 上传到服务器, 但是你会发现图片应该怎么传呢? 图片我使用了picgo生成一个图片链接, 把图片交给它会帮你生成一个链接, 通过这个链接访问你的图片, 这样不是更便捷吗? 而且我有存储桶, 阿里云几块钱租了一年, 将图片都存入其中, 那么在文件上传的途中, 可不可以将文件先通过picgo生成链接, 然后将链接存入到数据库当中, 从而实现文件上传, 然后又想了想, 那音频视频呢? 于是打算做一个简易版的音乐播放器, 文件上传作为音乐封面, 然后将音频也上传, 同时练练文件上传, 还有学习如何在vue进行picgo的使用, 从而做一个简易的音乐播放器, 这篇笔记用来记录一下这个过程, 理论可以, 让我们开始探索吧.....

## 创建Vue项目

首先确保你的电脑下载了Node.js和Vue cli, 现在都烂大街了, 因为我是mac就不带着下载了, 回头可以写一篇如何下载, 挖个坑回头填

1. 打开终端, 执行以下命令

```bash
vue create music-player
```

2. 选择你需要的特性

我只选择了vue3, 这都没什么, 后期需要可以在package.json当中配置一下

3. 切换到你的项目路径

```bash
cd music-player
```

4. 安装必要的依赖

```bash
npm install axios
```
